# KIJKO - Product Guide

## Product Vision

**KIJKO** is a unified Mission Control Center that provides a multi-modal interface for interacting with diverse AI models, combining agent orchestration capabilities with real-time observability.

## Target Users

### Primary Personas

1. **AI/ML Engineers**
   - Build and orchestrate AI agents
   - Need visibility into agent behavior and performance
   - Require seamless integration with multiple AI providers

2. **DevOps/Platform Teams**
   - Manage AI workloads at scale
   - Monitor system health and telemetry
   - Configure and maintain MCP integrations

3. **Software Developers**
   - Integrate AI tools into development workflows
   - Use conversational interfaces for coding assistance
   - Leverage multi-agent collaboration

## Core Mission

Provide a **unified multi-modal interface** that allows users to:
- Interact with AI agents through text, voice, and file attachments
- Monitor real-time agent activity and system health
- Discover and integrate Model Context Protocol (MCP) servers
- Manage context and conversation threads across agents

## Key Features

### 1. Agents Mode
- Multi-agent selection (Orchestrator, Coder, Reviewer, Researcher)
- Threaded conversations per agent
- Agent-specific role awareness

### 2. Hypervisa Mode (Hyperglyph)
- Multi-modal chat (text, voice, attachments)
- Context store for cached data
- Real-time typing indicators

### 3. Panopticon Mode
- MCP plugin status monitoring
- Live event telemetry stream
- Skill/capability discovery
- MCP marketplace browser

## Success Metrics

### Developer Experience
- **Onboarding Time**: < 5 minutes to first meaningful interaction
- **Task Completion**: Users can complete common tasks without documentation
- **Error Recovery**: Clear error messages with actionable guidance

### Feature Completeness
- **MCP Coverage**: Support for all major MCP servers
- **Agent Types**: Full support for all defined agent roles
- **API Integration**: Complete Gemini API implementation

## Product Principles

1. **Clarity over Cleverness**: Prioritize understandable interfaces over flashy features
2. **Fail Gracefully**: Always provide fallbacks and clear error states
3. **Performance Matters**: Sub-second response times for all UI interactions
4. **Extensibility**: Design for future MCP and agent integrations
